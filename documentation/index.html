Installation

First of all, thank you for purchasing this Bootstrap theme, your support makes it all possible!

This webpage takes you by hand on how to install this theme and will explain all technical details. If you are having difficulties when editing the theme I encourage you to read this page very thoroughly! You'll most likely find your answer here.

  1. Unzip the downloaded file tamashi-bamboo-1.0.0.rar to a new folder that you can name however you'd like. Your unzipped file will contain 2 folders. One is a folder named 'documentation' to guide you through customizing the theme and the second one is named 'theme' which is the actual one that holds all the files and folders you will be working with.
  2. Once your site is completed you will need to upload these files to your Web Server using FTP in order to use it on your website.
  3. Make sure you upload the required files/folders listed below (don't upload the documentation folder):
    • assets/ - Asset folder containing Bootstrap, LESS files, images, scripts and CSS
    • index.html - Theme HTML
    • style.css - Theme CSS
  4. Congratulations! Add your content and you'll be ready to launch your brand new website!

Page Structure

All the content will be wrapped by the body, a data-spy will be attached to update nav targets based on scroll position.

<body id="top" data-spy="scroll" data-offset="75" data-target=".navbar-fixed-top">
   ...
</body>

  1. Navbar/Header Nav
  2. Intro section #intro
  3. About section #about

  5. Portfolio section #portfolio
  6. Pricing section #pricing
  7. Features section #features
  8. Testimonials section #testimonials

  10. Footer Footer

Using LESS

Bamboo can be easily customized using LESS, all the theme specific files can be found in assets/less/theme. After you've editted your LESS files simple compile the file style.less and you will have a brand new customized CSS file!

Animations

For the animations we use wow.js. Here's a brief overview on how to modify or add/remove your own animations:

Make an element revealable

Add the CSS class .wow to a HTML element: it will be invisible until the user scrolls to reveal it.

<div class="wow">  Content to Reveal Here  </div >

Choose the animation style

Pick an animation style in Animate.css , then add the CSS class to the HTML element

<div class="wow bounceInUp">  Content to Reveal Here  </div >

Voila! The basics to get up and running with wow.js. Want more? Read the full documentation on wow.js here.

Fonts

Bootstrap utilizes 3 different fonts:

  1. Lato, the main font
  2. Montserrat, the font for the intro section heading
  3. Varela Round, used for sub-headings

These fonts can be edited or removed in the head section of the index.html and then in the bamboo-variables.less document. Located in assets/less/theme from line 45 and 47.

Colour Schemes

Bamboo consists out of 4 colours, 1 main colour, 1 secondary colour and 1 tertiary colour, paired with a 'cool gray'.

Primary / #96c93d

Secondary / #25aae1

Tertiary / #cb5c5a

Cool Gray / #212f3c

These colours are easily editable site-wide in the bamboo-variables.less document. Located in assets/less/theme from line 1 to line 18. Edit them to match your brand!

Logo

The logo can be modified in the navbar area of the theme:

<a class="page-scroll navbar-brand" href="#top"> Bamboo</a>

After changing the text you can modify the icon in the bamboo-navbar.less file. Located in assets/less/theme. Change the content value of navbar-brand:before with the value of the icon you'd like to use.

.navbar-brand {
  color: #fff;
  font-size: @navbar-bamboo-brand-size;
  font-family: @navbar-bamboo-brand-font-family;
  letter-spacing: -0.5px;
  line-height: 16px;
  &:hover,
  &:focus {
    background-color: transparent;
  }
  &:before {
    font-family: "outlined-iconset";
    content: "\e006"; //Change this value to swap icons
    vertical-align: middle;
    padding-bottom: 6px;
    color: @brand-primary;
  }
}

Click here for a complete list of available icons.

Using an image

If you already have a logo then feel free to replace the syntax with an image like so:

<a class="page-scroll navbar-brand" href="#top"><img src="assets/img/your-logo.png" class="img-responsive"></a>

Note: make sure you use a transparent .png image to make the logo blend in well with the navbar.

After inserting the image make sure you remove the CSS that generates the icon:

&:before {
  font-family: "outlined-iconset";
  content: "\e006"; //Change this value to swap icons
  vertical-align: middle;
  padding-bottom: 6px;
  color: @brand-primary;
}

Note: when using an image it's most likely that you'll have to adjust the padding of .navbar-brand.

Skewed links

The navigation links are skewed to match the theme, if you don't like it this way then you can find the necessary CSS to modify it in the bamboo-navbar.less file. Located in assets/less/theme on line 34 and 46. 

> li {
  transform: skewX(-27deg);  // Skew button to match theme
  border-right: 1px solid transparent;
  -webkit-transition: all .5s ease-in-out;
  -moz-transition: all .5s ease-in-out;
  transition: all .5s ease-in-out;
  &:nth-child(1) {
    border-left: 1px solid transparent;
  }
}
> li > a {
  color: fade(#fff, 50%);
  font-size: @navbar-font-size;
  transform: skewX(27deg); // Skew button to match theme, needs to be positive value of the li
  -webkit-transition: all .5s ease-in-out;
  -moz-transition: all .5s ease-in-out;
  transition: all .5s ease-in-out;
  &:hover,
  &:focus {
    color: #fff;
    background-color: @navbar-bamboo-link-hover-bg;
  }
}

Bamboo's navigation is skewed by 27 degrees as you can see in the example, edit it to fit your needs!

Backgrounds

Bamboo utilizes 3 kinds of backgrounds:

  1. The intro background
  2. Even background color
  3. Odd background color

Intro background

The intro background can be modifief in the bamboo-intro-section.less file. Located in assets/less/theme.

background: url(assets/img/theme-bg.jpg) no-repeat bottom center scroll;

Change the file theme-bg.jpg to modify the background.

Even background color

The background color of the even sections is #f6f8fa and can be modifief in the bamboo-global-section.less file. Located in assets/less/theme on line 35.

Odd background color

The background color of the odd sections is #ffffff and can be modifief in the bamboo-global-section.less file. Located in assets/less/theme on line 11.

Portfolio

The Bamboo portfolio is filterable, configure the categories in the .nav-pills div.

<ul class="nav nav-pills">
  <li class="active" data-group="all"><a href="#">All</a></li>
  <li data-group="branding"><a href="#">Branding</a></li>
  <li data-group="web"><a href="#">Web Design</a></li>
  <li data-group="logo"><a href="#">Logo Design</a></li>
  <li data-group="photography"><a href="#">Photography</a></li>
</ul>

After you've defined your data-groups you can add portfolio items in the #grid div according to the example, make sure your data-groups are the same!

<li class="col-md-3 col-sm-6 col-xs-6 portfolio-item" data-src="assets/img/portfolio/1-full.jpg" data-groups='[
"all", "branding"]'>
  <div class="portfolio-thumbnail">
    <div class="caption"><h4>Sleek UI Kit<br><small>Web Design</small></h4></div>
    <img class="img-responsive" src="assets/img/portfolio/1.jpg" alt="portfolio-item">
  </div> <!-- /.portfolio-thumbnail -->
</li>

Testimonials

Everyone knows that testimonials are a powerful way to let your visitors know about the work you did, and can often have a more powerful impact then a work example. So show off those testimonials! The testimonials are simple li's within a ul and you can add as much as you'd like:

<li class="testimonial">
  <blockquote>
    <div class="row">
      <div class="col-md-offset-4 col-md-4">
        <p>Lorem ipsum dolor sit amet, casdasdlandit, eget pretium nisl accumsan. Sed ultricies commodo tortoreu pretium mauris.</p>
        <small>Someone famous</small>
      </div>
    </div>
  </blockquote>
</li>

The p tag contains the testimonial and the small tag contains the name of the person who stated it.

Contact form

The Bamboo contact form is fully functional right out of the package, spiffy! Open the file processForm-contact.php located in the assets/email folder and edit these lines:

define( "RECIPIENT_NAME", "Your name" );
define( "RECIPIENT_EMAIL", "myemail@gmail.com" );
define( "EMAIL_SUBJECT", "Visitor Message" );

Google Maps

The map location and marker position can de edited in the assets/js/main.js file from line 30 to 43. 

// Init Google Maps
$(document).ready(function(){
  map = new GMaps({
    div: '#bamboo-map',
    lat: -12.043333,
    lng: -77.028333,
    enableNewStyle: true
  });
  map.addMarker({
    lat: -12.043333,
    lng: -77.028333,
    title: 'Location'
  });
});

Change the latitude and longitude to move the map around, these values can easily be found on Google.
